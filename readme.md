LanyardAPI for NodeJS and WebAPI

> To use in WEBAPI:
```html
raw.github/web/lanyardapi.js
```

> To use in NodeJS:
```js
// use by internet
const { LanyardAPI } = require('raw.github/web/lanyardapi.js')

// or download and use by path
const { LanyardAPI } = require('./lanyardapi.js')
```
